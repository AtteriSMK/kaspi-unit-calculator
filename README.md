# Kaspi Unit Calculator

Chrome-расширение (Manifest V3), встраивающее калькулятор unit-экономики прямо в карточку товара Kaspi.kz.

## Стек

- Vanilla TS + Vite + `@crxjs/vite-plugin` (HMR для расширений)
- Shadow DOM для изоляции стилей от Kaspi
- `chrome.storage.sync` — дефолты пользователя, `chrome.storage.local` — история по `product_id`
- Vitest для unit-тестов калькулятора

## Скрипты

```bash
npm install        # установка зависимостей
npm run dev        # dev-сборка с HMR (раздаётся в dist/)
npm run build      # production-сборка
npm test           # unit-тесты
npm run lint       # ESLint
```

## Установка распакованного расширения

1. `npm run build`
2. Открыть `chrome://extensions`
3. Включить «Режим разработчика»
4. «Загрузить распакованное» → выбрать папку `dist/`

## Важно про ставки

Hard-coded дефолты комиссий/логистики (12% комиссия, 799/1199/1599 ₸ логистика, 4% налог) — это разумные средние, не истина в последней инстанции. Реальные ставки Kaspi зависят от категории товара и типа договора — **сверяйте с актуальными в кабинете продавца** и переопределяйте через popup или прямо в карточке.
