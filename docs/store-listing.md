## Подробное описание (Description)

Поле в CWS: до 16000 символов.

```
Kaspi Unit Calculator — расширение для селлеров Kaspi Магазина. Считает unit-экономику прямо на странице товара: подставляет цену и минимального конкурента из карточки, моментально показывает прибыль, маржу, ROI, точку безубыточности и Δ к конкуренту по мере того, как вы меняете себестоимость и параметры.

ЧТО ЭТО РЕШАЕТ
• Перестать переключаться между карточкой Kaspi и Excel/Google Таблицами.
• Видеть на одном экране все составляющие unit-экономики: комиссию, логистику, ДРР, бонусы продавца, отзывной бонус, налог, упаковку.
• Сравнивать свою цену с минимумом по рынку прямо в момент анализа.

ОСНОВНЫЕ ФУНКЦИИ
• Виджет встраивается в правую колонку карточки товара — не закрывает контент Kaspi.
• Автоподстановка цены и минимального конкурента из данных карточки.
• Автоподбор тарифа логистики Kaspi по сумме заказа (тарифы на 01.01.2026 с НДС 12%).
• Мгновенный пересчёт прибыли, маржи, наценки, ROI и точки безубыточности.
• История расчётов по каждому товару — открыли карточку повторно, увидели сохранённые значения.
• Свернуть/закрыть виджет на конкретной карточке — он не вернётся, пока вы не очистите список скрытых.
• Дефолты для всех полей задаются в popup-настройках расширения.

ЧТО СЧИТАЕТ
• Прибыль = цена продажи − комиссия Kaspi − логистика − ДРР − бонусы продавца − бонус за отзыв − налог − себестоимость − упаковка.
• Маржа, наценка, ROI.
• Точка безубыточности (минимальная цена, при которой прибыль = 0).
• Δ к минимальному конкуренту (на сколько ваша цена выше/ниже).

ПРИВАТНОСТЬ
• Все данные хранятся локально в браузере (chrome.storage).
• Никакой телеметрии, аналитики, рекламы.
• Не передаём данные третьим сторонам.
• Не требуем регистрации.

Расширение независимое и не аффилировано с АО «Kaspi.kz».
```

## Single purpose statement

Поле в Privacy practices → Single purpose.

```
Расширение встраивает калькулятор unit-экономики в карточки товаров на kaspi.kz и сохраняет настройки и историю расчётов локально в браузере пользователя.
```

## Permission justifications

CWS требует объяснить каждое разрешение.

### `storage`
```
Хранит локально настройки калькулятора по умолчанию (себестоимость, комиссия, тарифы логистики, налог) и историю расчётов по карточкам товаров, чтобы пользователь не вводил значения заново при повторном открытии карточки.
```

### Host permission `https://kaspi.kz/shop/p/*`
```
Контент-скрипт встраивает виджет калькулятора в DOM страницы карточки товара Kaspi.kz и читает идентификатор, название и текущую цену товара из инлайн-данных самой страницы Kaspi.
```

### Remote code use
```
Нет. Все скрипты упакованы в расширение, никакой удалённый код не загружается и не исполняется.
```
