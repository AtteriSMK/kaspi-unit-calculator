Privacy Policy — Kaspi Unit Calculator

Last updated: 15 May 2026

1. Какие данные мы собираем

Расширение Kaspi Unit Calculator не собирает, не передаёт и не хранит персональные данные пользователя на внешних серверах. Все настройки и история расчётов сохраняются локально в браузере пользователя через Chrome API chrome.storage.sync и chrome.storage.local.

2. Что сохраняется локально

  • Пользовательские настройки калькулятора (себестоимость, комиссия, тарифы логистики, налог).
  • История расчётов по карточкам Kaspi.kz (ID товара, название, введённые значения).
  • Список скрытых карточек (товаров, на которых пользователь закрыл виджет).

3. Какие данные мы запрашиваем со страниц Kaspi.kz

На странице товара Kaspi.kz расширение читает из открытой страницы: идентификатор товара, название и текущую цену из инлайн-JSON, который Kaspi сам выгружает в HTML. Эти данные используются только для отображения калькулятора в браузере пользователя и не передаются никаким третьим сторонам.

4. Сторонние сервисы

Расширение не использует аналитику, рекламу, трекеры или внешние API, требующие передачи персональных данных. Не используются Google Analytics, Sentry, Mixpanel или аналоги.

5. Разрешения

  • storage — необходимо для сохранения настроек и истории расчётов локально в браузере.
  • Host match https://kaspi.kz/shop/p/* — необходимо для инжекта калькулятора в карточки товаров Kaspi.kz.

6. Удаление данных

Удалить все данные расширения можно одним из способов:

  • Удалить расширение из chrome://extensions/ — Chrome автоматически очистит storage.
  • В настройках расширения нажать «Сбросить» — настройки вернутся к заводским.

7. Изменения политики

Если политика изменится, обновление публикуется на этой же странице с новой датой «Last updated». Существенные изменения сопровождаются обновлением версии расширения в Chrome Web Store.
